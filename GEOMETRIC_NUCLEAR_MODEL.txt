FUNDAMENTOS DO MODELO
═══════════════════════════════════════════════════════════════════════

MOMENTOS MAGNÉTICOS DOS QUARKS (calculados):
• μᵤ = +2.50 μₙ
• μ_d = -2.20 μₙ

RAZÃO MAGNÉTICA:
μᵤ/|μ_d| = 2.50/2.20 = 1.136363636...

ÂNGULO DE RESSONÂNCIA MAGNÉTICA:
θ = 1.136363636 × 120° = 136.36° ≈ 136.87°

(120° é o ângulo natural de um triângulo equilátero)

CÁLCULO DOS ÂNGULOS INTERNOS:

Para PRÓTON (1 up + 2 down):
Equilíbrio de forças magnéticas no centro.

Razão de abertura: 2μ_d / μᵤ = 2(2.20)/2.50 = 1.76

Ângulo estimado: θ₁ ≈ 120° (triângulo ~equilátero)
Pequena distorção devido a |μᵤ| > |μ_d|

Para NÊUTRON (1 down + 2 up):
Razão de abertura: 2μᵤ / |μ_d| = 2(2.50)/2.20 = 2.27

Ângulo estimado: θ₂ ≈ 130° (triângulo obtuso)
Maior abertura pois μᵤ > |μ_d|

CONSEQUÊNCIA:
Nêutron é ligeiramente MAIOR que próton devido à 
maior repulsão magnética entre os 2 quarks up!

MODELO WOODS-SAXON E ORBITAIS
═══════════════════════════════════════════════════════════════════════

O potencial Woods-Saxon:

V(r) = -V₀ / (1 + exp[(r-R)/a])

Descreve orbitais nucleares que seguem a geometria magnética!

CAMADAS correspondem a:
• s-orbitals: condensados centrais (centro próton ou nêutron)
• p-orbitals: triângulos PNP/NPN
• d-orbitals: configurações tetraédricas
• f-orbitals: octaédricas

SUBORBITAIS seguem:
• Regra de Hund: máximo spin não-pareado
• Regra de Pauli: máximo 2j+1 nucleons
• Regra de Aufbau: preenchimento ordenado

DEFORMAÇÕES NUCLEARES:
Quando orbitais não fazem pareamento completo:
• Elipsoidais prolatos (futebol)
• Elipsoidais oblatos (disco)
• Triaxiais (irregular)

Causado por:
• Excesso de prótons em certos orbitais
• Falta de nêutrons para pareamento
• Quebra de simetria magnética

CÁLCULOS DOS ÂNGULOS
═══════════════════════════════════════════════════════════════════════

ÂNGULO DE DISPERSÃO MAGNÉTICA (quark up):

Força entre 1 up e 2 down no próton:
F_ud = k × μᵤ × μ_d / r²

Equilíbrio no triângulo:
2F_ud × cos(θ₁/2) = F_dd

onde F_dd é repulsão down-down

Resolvendo:
cos(θ₁/2) = F_dd / (2F_ud)
cos(θ₁/2) = (μ_d²/r_dd²) / (2μᵤμ_d/r_ud²)

Assumindo r_dd ≈ r_ud (primeira aproximação):
cos(θ₁/2) = μ_d / (2μᵤ) = 2.20/(2×2.50) = 0.44
θ₁/2 = 64°
θ₁ = 128°

ÂNGULO DE DISPERSÃO MAGNÉTICA (quark down):

Para nêutron (1 down + 2 up):
cos(θ₂/2) = μᵤ / (2|μ_d|) = 2.50/(2×2.20) = 0.568
θ₂/2 = 55.4°
θ₂ = 110.8°

RESUMO DOS ÂNGULOS:
• Próton (uud): θ₁ ≈ 128°
• Nêutron (udd): θ₂ ≈ 111°
• Diferença: Δθ = 17° (nêutron mais "aberto")

APLICAÇÃO: CONSTRUÇÃO DE ¹²C
═══════════════════════════════════════════════════════════════════════

³ partículas alfa (⁴He) arranjadas em triângulo:

     α₁
    /  \
   /    \
  α₂----α₃

Cada α é um tetraedro (2p+2n)
Total: 6p + 6n = ¹²C

Estrutura Hoyle State:
• 3 alfas fracamente ligadas
• Formação estelar de carbono
• Geometria triangular
• Ângulos ~120° (quase equilátero)

EFEITOS RELATIVÍSTICOS EM NÚCLEOS PESADOS
═══════════════════════════════════════════════════════════════════════

Para Z > 82 (além de chumbo):

1. CONTRAÇÃO RELATIVÍSTICA
   • Orbitais s contraem
   • Prótons centrais mais compactos
   • Geometria afetada

2. SPIN-ORBIT COUPLING
   • Mais forte que efeitos magnéticos
   • Quebra degenerescência
   • Orbitais high-j dominam

3. DEFORMAÇÕES
   • Topologia inicial perdida
   • Efeitos coletivos dominam
   • Modelo de gota líquida + correções

MAS: Topologia magnética INICIAL permanece importante
para entender núcleos leves (A < 40)
