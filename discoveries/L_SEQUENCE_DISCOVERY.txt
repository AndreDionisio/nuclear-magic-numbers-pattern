EXPLICAÇÃO DO PADRÃO +4
═══════════════════════════════════════════════════════════════════════

O incremento "+4" na fórmula fenomenológica NÃO é arbitrário!

Ele captura o seguinte padrão físico:

1. Entre números mágicos maiores (50+), os orbitais preenchidos
   têm valores de l que seguem NÚMEROS ÍMPARES: 1, 3, 5, 7, 9...

2. Para orbitais de alto-j (j = l + 1/2), a capacidade é:
   c = 2j + 1 = 2(l + 1/2) + 1 = 2l + 2

3. Progressão de capacidades para l ímpar:
   l=1: c=4
   l=3: c=8
   l=5: c=12
   l=7: c=16
   l=9: c=20
   
   Incremento: +4 consistente!

4. Portanto, o "+4" na fórmula reflete o SALTO de um número
   ímpar para o próximo (+2 em l), que se traduz em +4 na
   capacidade devido à relação c = 2l + 2.

POR QUE NÚMEROS ÍMPARES DOMINAM?
═══════════════════════════════════════════════════════════════════════

A dominância de orbitais com l ímpar em números mágicos pesados
reflete o ACOPLAMENTO SPIN-ÓRBITA FORTE:

1. Spin-órbita divide cada orbital l em dois: j = l ± 1/2

2. Para l grande, o orbital j = l + 1/2 tem capacidade muito
   maior e energia mais baixa que j = l - 1/2

3. Números ímpares de l (1,3,5,7,9) correspondem aos orbitais
   p, f, h, j... que têm os maiores valores de j quando somados
   com spin +1/2

4. Estes orbitais de alto-j são preenchidos PRIMEIRO devido ao
   forte acoplamento spin-órbita
