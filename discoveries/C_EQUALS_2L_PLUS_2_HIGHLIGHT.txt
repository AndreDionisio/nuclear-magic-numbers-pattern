A RELAÇÃO FUNDAMENTAL
═══════════════════════════════════════════════════════════════════════

A capacidade de um orbital de alto-j é dada por:

    c = 2j + 1

Para orbitais onde j = l + 1/2 (spin-órbita positivo):

    c = 2(l + 1/2) + 1 = 2l + 1 + 1

    ╔═══════════════╗
    ║  c = 2l + 2   ║
    ╚═══════════════╝

Esta é a equação-chave que conecta:
• A capacidade orbital (c) ← parâmetro da fórmula fenomenológica
• O momento angular orbital (l) ← número quântico fundamental

═══════════════════════════════════════════════════════════════════════
POR QUE ISTO EXPLICA O +4
═══════════════════════════════════════════════════════════════════════

PROGRESSÃO DE l ÍMPAR:

l = 1, 3, 5, 7, 9... (p, f, h, j...)

PROGRESSÃO DE CAPACIDADES:

l=1: c = 2(1) + 2 = 4
l=3: c = 2(3) + 2 = 8   ← diferença: +4
l=5: c = 2(5) + 2 = 12  ← diferença: +4
l=7: c = 2(7) + 2 = 16  ← diferença: +4
l=9: c = 2(9) + 2 = 20  ← diferença: +4

MATEMÁTICA DO INCREMENTO:

Quando l aumenta em +2 (de um ímpar para o próximo):
    Δl = +2

A variação em c é:
    Δc = 2(Δl) = 2(2) = 4

Portanto:
    ╔═════════════════════════════════╗
    ║  Δc = 2Δl = 2(2) = +4           ║
    ║                                 ║
    ║  "O +4 vem de Δl = +2"         ║
    ╚═════════════════════════════════╝
